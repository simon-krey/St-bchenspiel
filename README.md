
# Stäbchenspiel

Ein Projekt für den Informatikunterricht.

## Run Locally

Clone the project

```bash
  git clone https://github.com/simon-krey/St-bchenspiel.git
```

Go to the project directory

```bash
  cd St-bchenspiel
```

Install dependencies

```bash
  pip install -r requirements.txt 
```

Start the application

```bash
  python main.py
```
